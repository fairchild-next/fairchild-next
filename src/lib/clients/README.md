# Client-specific content

- **`fairchild/weddingContent.ts`** – URLs, copy, and image paths for Wedding Mode. For another venue, add `clients/<slug>/weddingContent.ts` with the same exports and point UI imports at that module (or use a small registry keyed by env).

Images for Fairchild wedding mode live in **`public/wedding/`** (see filenames in `weddingContent.ts`).
